# ansible-java-ubuntu

# Description

**ansible-java-ubuntu** is a simple [Ansible](https://www.ansible.com/) role for installing Java on an Ubuntu machine.

# Usage

Within Ansible playbook:

```
...
roles:
  - ansible-java-ubuntu
...
```
